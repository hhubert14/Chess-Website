# Chess Openings Explorer

A web-based platform for learning and exploring chess openings through interactive tutorials and visual board demonstrations.

## Features

- **Interactive Tutorials** — Step-by-step guides for various chess openings with interactive boards to practice moves
- **Opening Trees** — Detailed opening trees from classic to modern variations
- **Analysis Tools** — Positional and tactical analysis with insights and recommendations

## Supported Openings

- King's Indian Defense
- King's Indian Attack
- Pirc Defense

## Tech Stack

- HTML, CSS, JavaScript
- Interactive chessboard rendering
- Move validation and game state management

## Getting Started

1. Clone the repository
2. Open `index.html` in a browser
3. Browse the supported openings and follow the interactive tutorials
